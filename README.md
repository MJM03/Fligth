# AeroRadar v1.0.0

Demo web/PWA de seguimiento de tráfico aéreo, preparado para GitHub Pages.

## Publicación
1. Crea un repositorio público en GitHub.
2. Sube todos los archivos de este ZIP directamente a la raíz, sin crear carpetas.
3. Abre Settings > Pages.
4. En Source selecciona Deploy from a branch.
5. Selecciona main / root y guarda.

## Datos
- Fuente principal: ADSB.lol API pública.
- El mapa consulta aeronaves alrededor del centro del área visible.
- Si la API no responde, la interfaz activa automáticamente un modo demostración.
- Los datos ADS-B no garantizan ruta comercial, origen, destino ni horarios.
